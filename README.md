## 💻 Sobre o projeto

https://matheusalvarez.github.io/Conversor_RGB-Hex/

Apresento um projeto que simplifica a conversão de valores de cores RGB para Hexadecimal, criado com HTML, CSS e JavaScript. Este conversor fornece uma maneira fácil e intuitiva de transformar valores de cores, facilitando a vida de designers e desenvolvedores que trabalham com paletas de cores na web.
